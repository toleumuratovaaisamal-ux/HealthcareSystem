# Healthcare Management System

Simple Java project for managing patients, medical professionals and hospital.

## Features
- Object-Oriented Programming (classes, constructors, methods)
- Encapsulation with private fields and setters/getters
- Data pool using ArrayList
- Filtering admitted patients
- Overridden toString() methods

## How to run
1. Open project in IntelliJ IDEA
2. Run Main.java
